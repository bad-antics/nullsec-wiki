# 🏠 NullSec Wiki Home

Welcome to the official NullSec Security Framework documentation. This wiki covers everything from installation to advanced tool development.

## What is NullSec?

**NullSec** is a comprehensive security framework consisting of:

- **NullSec Linux v5.0** — A security-focused Linux distribution with 140+ pre-installed tools
- **NullKia v3.0** — Mobile security framework supporting 18 device manufacturers
- **Specialized Tool Suites** — Cloud, AI/ML, Hardware, and Automotive security tools
- **706+ Open Source Repositories** — Tools across 15+ programming languages

## Core Components

### 🐧 NullSec Linux

A Debian 13-based security distribution designed for:
- Penetration testing
- Security research
- Vulnerability assessment
- Digital forensics
- Incident response

**Features:**
- 140+ security tools pre-installed
- 5 specialized editions
- 4 architecture support (AMD64, ARM64, RISC-V, Apple Silicon)
- Kernel 6.8 LTS with security hardening
- Wayland + Hyprland compositor

### 📱 NullKia Mobile Security

A cross-platform mobile security framework for:
- Device security research
- Baseband exploitation
- Cellular network analysis
- TEE/TrustZone research
- Firmware extraction

**Features:**
- 18 manufacturer support
- Baseband tools (Shannon, Qualcomm, MediaTek)
- 5G/LTE security testing
- eSIM management
- Plugin architecture

### ☁️ Cloud Security Suite

Tools for cloud infrastructure security:
- AWS reconnaissance
- GCP hunting
- Azure sweeping
- Kubernetes scanning
- Terraform auditing

### 🤖 AI/ML Security Suite

Tools for AI/ML security research:
- LLM red teaming
- Prompt injection testing
- Model auditing
- Adversarial examples
- Data poisoning detection

### 🔌 Hardware Security Suite

Tools for hardware hacking:
- SDR analysis
- RFID exploitation
- CAN bus testing
- JTAG debugging
- Voltage glitching

### 🚗 Automotive Security Suite

Tools for vehicle security:
- CAN bus analysis
- OBD-II diagnostics
- UDS protocol testing
- Key fob reverse engineering

## Quick Start

1. **Download NullSec Linux** from [releases](https://github.com/bad-antics/nullsec-linux/releases)
2. **Verify the ISO** with GPG signature
3. **Boot** from USB or install to disk
4. **Explore tools** organized by category in the applications menu
