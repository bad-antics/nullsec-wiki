# 📝 Blog Post Template

*Use this template when creating new daily blog posts.*

## Naming Convention

```
Blog-YYYY-MM-DD-Short-Title.md
```

Example: `Blog-2026-03-07-Log-Analysis-IR.md`

## Template

```markdown
# [Emoji] Title — Subtitle

*[Date] • bad-antics • #tag1 #tag2 #tag3*

---

## Opening Hook

[Start with a scenario, question, or bold statement that draws the reader in]

---

## Main Content

### Section 1: Context/Problem
[What are we talking about and why does it matter?]

### Section 2: Technical Deep Dive
[The meat — code, logs, configs, analysis]

### Section 3: What Others Miss
[Your unique insight or angle]

---

## Practical Takeaways

1. [Actionable item 1]
2. [Actionable item 2]
3. [Actionable item 3]

---

## Tomorrow

[Tease the next post topic]

---

*All content for authorized security testing and educational purposes only.*

**Tags:** `tag1` `tag2` `tag3`

---

← [Blog Index](Blog-Index) • [Previous Post →](Previous-Post-Link)
```

## Checklist Before Publishing

- [ ] Post filename follows `Blog-YYYY-MM-DD-Short-Title.md` convention
- [ ] Added entry to [Blog-Index.md](Blog-Index.md) in the monthly table
- [ ] Added to appropriate topic section in Blog-Index.md
- [ ] Updated Blog-Index.md post count stat
- [ ] Updated Home.md blog section if it's a major post
- [ ] Cross-referenced any NullSec tools mentioned
- [ ] Tags are relevant and consistent
- [ ] Tomorrow teaser aligns with planned next post
